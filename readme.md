## 문자와 관련된 내용 정리 ( 천천히 정리하기 🤓 )

CR, LF

- \r (CR) : carriage return (cursor to beggining of the line)
- \n (LF) : line feed (cursor to next line)
- \r\n (CR + LF) : end of line ()
